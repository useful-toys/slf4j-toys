## Upgrade Mockito and Maven Plugins

This PR modernizes the build system with critical updates for Java 21 compatibility and enforces consistent build environment across teams.

### Changes

#### 1. Mockito 5.7.0 Upgrade (Java 21 Compatibility)
- **Updated:** Mockito from 4.11.0 → 5.7.0
- **Benefit:** Resolves 103 test failures caused by Java 21 incompatibilities
- **Impact:** 98% error reduction (103 → 2 errors remaining, unrelated to Mockito)

#### 2. Maven Version Enforcement
- **Added:** maven-enforcer-plugin 3.6.2
- **Minimum version:** Maven 3.9.8 (matches wrapper version)
- **Benefit:** Ensures consistent build environment across all developers

#### 3. Maven Plugin Updates
All core plugins updated to latest versions compatible with Maven 3.6.3+:
- maven-compiler-plugin: 3.13.0 → **3.14.1**
- maven-resources-plugin: 3.3.1 → **3.4.0**
- maven-surefire-plugin: 3.5.3 → **3.5.4**
- maven-source-plugin: 3.3.1 → **3.4.0**
- flatten-maven-plugin: 1.5.0 → **1.7.3**
- versions-maven-plugin: 2.18.0 → **2.20.1**

#### 4. Plugin Version Centralization
- **Added:** Properties for all Maven plugin versions in parent POM
- **Updated:** All hardcoded plugin versions to use `${property.version}` format
- **Benefit:** Single source of truth for plugin versions, easier maintenance
